# dc-apply

Career search operating system for Dev Chandra Patel.

Primary goal: find and win the next role, with a focus on Head of Growth, Founding Operator / Chief of Staff, and Product Management Leader opportunities in AI, defense tech, gov tech, and deep tech.

## Operating mode

Current mode: **prepare for review**.

Hermes may:
- Research and rank roles.
- Add roles and tasks to Notion.
- Draft tailored resumes, pitches, and application materials.
- Prepare applications for Dev's review.

Hermes should not submit applications autonomously yet. Move to autonomous mode later after quality is established.

## Source of truth

- Notion project: `dc-apply`
- GitHub repo: supporting files, prompts, resume variants, pitches, and scripts
- Master resume: `profile/master-resume.md`
