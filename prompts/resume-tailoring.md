# Resume Tailoring Prompt

Using `profile/master-resume.md`, tailor a resume for {{company}} — {{role}}.

Constraints:

- Preserve truthfulness.
- Keep one-page executive-style resume unless otherwise requested.
- Prioritize metrics and role-specific relevance.
- Match the role's language without keyword stuffing.
- Produce both Markdown and notes on what changed.

Return:

1. Tailored summary
2. Revised experience bullets only where needed
3. Skills/keywords to include
4. Potential cover-letter angle
